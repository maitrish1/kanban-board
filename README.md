# Kanban Board

Welcome to the Kanban Board, a simple and efficient application built with React and TypeScript to streamline your project management process.

## Features

- **Drag-and-Drop Functionality:** Utilizing the power of the `react-beautiful-dnd` library, enjoy seamless drag-and-drop functionality for both tasks and columns.

- **Inline Editing:** Edit tasks and columns on the fly with inline editing for content and titles, enhancing flexibility and convenience.

- **Responsive Design:** Experience a user-friendly interface that adapts to both desktop and mobile devices, ensuring productivity on the go.

## Installation

Follow these simple steps to get started with the Kanban Board:

1. Clone the repository:
   ```
   git clone https://github.com/maitrish1/kanban-board.git
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

## Usage

Get the most out of the Kanban Board with these easy-to-follow usage instructions:

1. **Create a New Task:**
   Click the "Add Task" button to initiate the creation of a new task.

2. **Task Management:**
   - **Drag and Drop:** Change the status of tasks by dragging and dropping them between columns.
   - **Inline Editing:** Click on a task to edit its content or title.

3. **Delete Tasks and Columns:**
   Easily remove a task or column by clicking the "Delete" button.

Feel free to tailor the Kanban Board to your project management needs and enjoy a smoother workflow.
